## For totally new use
git clone https://github.com/thunderking211/ILMS.git
git config user.name "Your Name"
git config user.email "your Github Email ID"
git config credential.helper cache

## If you already have project on your desktop and didn't initialized it..
git init
git add remote origin https://github.com/thunderking211/ILMS.git
git config user.name "Your Name"
git config user.email "your Github Email ID"
git config credential.helper cache
git remote update
git pull --rebase

## If you already have both repo and git set 
git remote update
git pull --rebase
